Installation

Add styles and scripts to your page:

<script src="/path/to/{{name}}.min.js"></script>
<link href="/path/to/{{name}}.css" rel="stylesheet" />
  • Don't forget to load jQuery before {{name}}.min.js!
  • {{name}}.js must be put inside the body tag.

Usage

Basic notification:

$.Motice({ text: 'Hello, world!' });
Run

Notification with image:

$.Motice({ text: 'Hello, image!', image: 'https://octodex.github.com/images/privateinvestocat.jpg' });
Run

Notification on top right corner:

$.Motice({ text: 'Notification at top right', position: 'right-top' });
Run

Notification with auto width:

$.Motice({ width: 'auto', text: 'Not too long text' });
Run

Notification on top with fluid width:

$.Motice({ position: 'fluid-top', text: 'Notification at top with fluid width' });
Run

Notification on bottom with fluid width:

$.Motice({ position: 'fluid-bottom', text: 'Notification at bottom with fluid width' });
Run

Notification with jelly effect:

$.Motice({ autoHide:false, effect: 'jelly', text: 'Notification with jelly effect' });
Run

Options

All Motice options:

{
  // Image path (http/base64)
  image: undefined,
  // Position on screen, values: right-bottom, right-top, left-bottom, left-top, fluid-top, fluid-bottom
  position: 'right-bottom',
  // Theme
  theme: 'dark',
  // Template, these classes should ALWAYS be there
  template: '
',
  // css width
  width: '300',
  // Text that will be displayed in notification
  text: '',
  // Display background or not, if false, background: transparent;
  background: true,
  // Hide automatically
  autoHide: true,
  // Hide by click
  clickHide: true,
  // Autohide delay
  delay: 3500,
  // Enable animation effects, default is 'default', another effect called 'jelly' is also provided inernally.
  effect: 'default',
  // We will put an effect class to the notification dom combining 'effectPrefix' and 'effect'
  effectPrefix:'motice-effect-'
};

Themes

Default themes:

Dark (default)
Light

Creating custom themes:

Example:

$.Motice.addTheme('dark', {
    classes: 'motice-theme-dark'
});
.motice-theme-dark {
  background-color: #202020;
  color: #f5f5f5;
}

Options:

{
  /*

    Blocks order:

    | container
    | - base
    | -- image
    | -- text
    | - base
    | -- image
    | -- text
    | ...

  */

  // Classes that will be added to base element

  classes: ''
}

Custom animation effects

You can use your own show and hide css3 animations. If user's browser does not support css3 animations, simple hide/show will be used as fallback.

CSS 

@keyframes custom-hide-animation {
  0% {
    transform: scale(1, 1);
  }
  20% {
    opacity: 1;
  }
  100% {
    opacity: 0;
    transform: scale(1.5, 1.5);
  }
}
@keyframes custom-show-animation {
    from {
        opacity:0;
    }
    to {
        opacity:1;
    }
}
.motice-effect-custom.motice-hide {
    animation: custom-hide-animation 0.5s forwards;
}
.motice-effect-custom.motice-show {
    animation: custom-show-animation 0.5s forwards;
}

JavaScript 

$.Motice({
  text: 'Custom hide animation',
  effect: 'custom'
});
Run

Build

To build Motice from sources you need to install Gulp 

$ npm install gulp -g

Then navigate in terminal to downloaded sources and install some required modules 

$ npm install

Then run Gulp 

$ gulp

You can take compressed and optimized files from dist folder
